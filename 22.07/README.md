# GIT TUTORIAL

## git init
___
* git 저장소 생성  

## git add
___
* stage area에 업데이트. untracked 상태면 tracked 상태로 변경

## git commit
___
* repository에 버전 기록하기
* -m옵션으로 메시지 입력

## git push
___
* 원격 저장소에 로컬 저장소 내용 update하기
* 처음 push할 때 -u origin master 를 통해 원격 저장소의 master 브랜치와 연결이 필요함  

## git clone 
___
* 원격 저장소 내용 가져오기
* 원격 저장소 이름으로 폴더가 생김  

## git remote add origin 
___
* 로컬저장소와 원격 저장소 연결하기
